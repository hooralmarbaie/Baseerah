# Baseerah – Arabic Braille Reading Assistant

## Overview

Baseerah is a web-based Arabic Braille reading assistant that converts Arabic text, scanned images, and PDF documents into Unified Arabic Braille using AI-powered text extraction and verification followed by a rule-based Braille translation engine.


## Features

- Supports Arabic text, scanned images, and PDF documents
- AI-powered text extraction and verification
- Rule-based Unified Arabic Braille translation
- Back translation validation
- Accessible web interface for visually impaired users


## Technologies

- Python
- Flask
- HTML/CSS/JavaScript
- GPT-4o
- PyMuPDF
- RapidFuzz
- Pandas


## Workflow

Input → Text Extraction → AI Verification → Text Processing → Rule-Based Braille Translation → Back Translation → Evaluation → Output


## Results

- Supports multiple Arabic input formats
- Reliable Arabic text extraction and verification
- Accurate Unified Arabic Braille translation
- Evaluated using CER, WER, and similarity metrics
